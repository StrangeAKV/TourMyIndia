
Tour My Country

A modern travel booking and exploration website that allows users to discover destinations, book trips, and manage travel plans seamlessly. This project is inspired by platforms like MakeMyTrip and provides an intuitive UI for travel-related services.

✨ Features
🏨 Hotel Booking – Explore and reserve hotels easily
🌴 Holiday Packages – Discover curated travel experiences
🔐 User Authentication – Login/Signup functionality
🎁 Offers & Deals Section – Special discounts and promotions
📱 Responsive Design – Works smoothly on mobile & desktop
